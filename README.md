# vant-app

1. .env环境变量配置
2. vue.config.js 路径别名，代理设置
3. vant按需加载以及移动端rem配置postcss.config.js
4. vuex配置，包含工具插件vuex-loading获取数据加载插件，vuexlocal本地存储插件

yarn install
